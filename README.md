# Barcode-app

Created to help move away from Excel-based barcode credential creation

---

## How to run

### Git clone repo

```bash
git clone https://github.com/LLosTy/barcode-app
```

### Navigate to repo

```bash
cd Barcode-app/
```

### Install using

```
npm i
```

### Run

```
npm run dev
```

### Enjoy
