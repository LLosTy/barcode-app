# Barcode-app

Created to help move away from Excel-based barcode credential creation

---

## How to run

### Clone git repo

```bash
git clone git@github.com:LLosTy/barcode-app.git
```

### Navigate to repo

```bash
cd Barcode-app/
```

### Install using

```
npm i
```

### Run

```
npm run dev
```

### Enjoy
